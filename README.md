# sequoyah-elder-risk-capstone
Reproducible GIS capstone workflow for a synthetic elder-risk dashboard in Sequoyah County, Oklahoma.

This repository contains scripts, markdown documentation, validation outputs, and dashboard configuration notes for a GIS capstone project. It does not include the final paper, presentation slides, full geodatabase, ArcGIS Pro project file, hosted ArcGIS Online layer, or any real person-level records.
